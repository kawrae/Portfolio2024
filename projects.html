Horror Game Alpha V1.0

Download

Horror Game Alpha V1.0 is an unfinished first-person horror experience where players navigate through a dark, eerie environment.

Features:

  • Dynamic lighting and realistic sound effects to enhance the atmosphere.
  • Jumpscare trigger technology as well as an interactive inventory system.
  • Object interaction, advanced movement, and level progression.

This is an early alpha version, so there may be bugs and the game is far from completion. This project served as a general insight into techniques using C# and familiarizing myself with the Unity Engine UX. Feedback is welcome!
